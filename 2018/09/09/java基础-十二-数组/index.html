什么是数组

通过我们之前的学习，要储存一个人的成绩，如果它是整型的话，我们可以用 int score.
那如果我们要储存一个班的成绩呢？这个时候就必须使用到数组了。
数组就是将同一个类型的一系列数归为一组的数据类型。

数组在初始化后长度就固定了。如果要对数组长度进行修改，称之为扩容和缩容。
扩容和缩容实际上就是新建一个对应容量的数组，将当前数组中的元素拷贝到新数组中，然后将当前数组地址指向新数组,比如以下为arr扩容一个长度:
