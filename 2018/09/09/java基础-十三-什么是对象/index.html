toString

toString()方法是java为对象设计的转换为字符串的默认方法。System.out.println()在传入一个自定义类的对象时，会去调用这个对象的toString （）方法。如果没有，则会输出这个对象的地址。

对象的引用和实际值

在这个基本的声明中，new 对象类型()新建了一个对象，但实际上是返回了一个地址。而对象名中储存的就是这个地址。在我们调用对象名的时候，是通过变量里面储存的地址找到真正的对象并对其操作。
我们将公式中的对象名叫做引用，而 new 对象类型() 才是新建了实际值。
