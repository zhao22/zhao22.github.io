什么是方法

方法的结构

java中创建一个方法应满足以下结构:

修饰符

修饰符 意义
public 该方法的调用不受限制。
(不填) 该方法只能在当前package下调用。
protected 该方法只能在本类和子类中调用。(子类的概念在介绍面向对象时介绍)
private 该方法只能在本类调用

我们应该考虑方法的用途，为方法选择尽量窄的修饰符。比如一个方法只是在类的内部调用。我们应该选用private作为方法的修饰符，这样方法的功能会更加明确，项目的结构会更加清晰。

static

加上static时，表示这个方法是静态方法，是属于这个类的。
不加static时，是属于这个类实例化后的对象的。

比较抽象，将会在介绍面向对象后专门介绍，暂时写方法请使用static修饰的静态方法。

返回值类型

返回值类型是一个类名，如果声明了某一种返回值类型，则这个方法必须在结束时返回一个属于这个类型的变量。
返回的变量可以被调用方使用。
如果没有返回值，返回值类型填写void。

方法的调用

如果这个方法是静态方法:
我们可以使用 类名.方法名(变量1，变量2…)来调用。
如果方法有返回值,我们可以用一个和返回值类型匹配的参数来接受：
返回值类型 变量 = 类名.方法名(变量1，变量2…)

如果这个方法不是静态方法:
我们需要使用 对象.方法名(变量1, 变量2…)来调用。
和静态方法一样，在有返回值时，我们可以使用变量来接收。

如果被调用方法就是当前类的方法：
我们可以直接使用方法名(变量1，变量2…)来调用。

但要注意,静态方法不能直接调用非静态方法。
