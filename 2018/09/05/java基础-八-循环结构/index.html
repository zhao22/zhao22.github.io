do..while..循环

如果while可以翻译成：当满足一定条件的话，循环执行循环体中的代码。
那么do…while就可以翻译成: 执行循环体中的代码，当满足一定条件的时候进行循环。
它和while循环的区别是：
如果不满足条件，while循环内的循环体一次都不会执行。
而do…while循环无论条件满不满足，都会执行一遍循环体。

System.out.println

System.out.println我们前面用得比较熟悉了,只要调用它，在括号里面填写相应的字符串，我们就能在控制台打印出它的内容。
System.out 是一个专门在控制台上打印东西的打印流。println只是它的一个方法，读法是 print line。
意思是它会将接下来的一行用于打印括号内的字符串,多次打印的话就会换行。
