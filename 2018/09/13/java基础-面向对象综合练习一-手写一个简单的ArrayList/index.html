什么是ArrayList

在日常的使用数组的过程中。如果需要存的数据长度是不确定的，我们时常会需要扩容。如果要删除某个元素，我们则需要对数组进行缩容。是比较不方便的。
我们可以新建一个类，这个类将数组封装在内部储存数据。提供方法使这个类可以和数组一样初始化，获取元素，新增元素，修改元素，遍历元素。同时将扩容封装在类的内部，在调用时不用关心扩容与否，删除元素也将缩容封装在内部，能较大地提升我们的开发效率。
